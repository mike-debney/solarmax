# SolarMax Integration

Estimates solar panel power output based on solar radiation, panel specifications, and orientation using the pvlib library.

## Features

- Configure multiple solar arrays with different orientations
- Individual power sensors for each array plus total output
- Real-time calculations using solar position and angle-of-incidence
- Configurable inverter efficiency and capacity
- Updates automatically when solar radiation changes
